

## Aesthetic Enhancement Suggestions for SeekLakaw

The app already has a strong visual foundation with the #ec407a pink palette, glassmorphism, Dancing Script branding, confetti particles, and animated elements. Here are targeted improvements to elevate it further:

### 1. Bottom Navigation Bar
Add a fixed bottom tab bar (Home, Calendar, Favorites, Profile) with the pink accent color, a floating active indicator, and subtle haptic-feel animations. This replaces the top-right icon buttons and gives the app a polished native-mobile feel.

### 2. Parallax Hero Header
Make the hero gradient header respond to scroll — shrink the title, fade the sparkle dots, and collapse into a compact sticky toolbar as the user scrolls down. This adds depth and a premium interaction feel.

### 3. Skeleton Loading States
Replace the current loading state with animated skeleton placeholders (shimmer effect) that match the card layout. This eliminates the jarring content pop-in and feels more refined.

### 4. Image Lazy Loading with Blur-Up
Show a tiny blurred placeholder while festival images load, then transition to the full image with a smooth fade. This prevents empty white boxes and adds a magazine-quality feel.

### 5. Micro-Interaction Enhancements
- Heart favorite button: burst particle animation on tap (tiny pink dots radiating outward)
- Category pills: elastic spring animation when switching
- Pull-to-refresh gesture with a custom pink spinner

### 6. Section Dividers with Decorative Elements
Add subtle Filipino-inspired decorative dividers (e.g., a thin line with a small sun/star motif in the center) between sections instead of plain spacing.

### 7. Dark Mode Polish
If not already refined, ensure the dark mode uses deep charcoal backgrounds with the pink accent glowing subtly — not just inverted colors.

---

**Recommendation:** Start with the **bottom navigation bar** — it has the highest impact on perceived app quality and usability, especially on mobile.
